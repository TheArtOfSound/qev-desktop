Honest scope.

The inline app is convenient. The CLI remains the cleaner production workflow because browser pages can inherit browser, extension, and CDN risks.

QEV does not protect against a compromised device, weak phrase, malicious extension, CDN compromise, or forgotten phrase.

Use the CLI for stronger local control. Use this page for fast public demo, simple vault use, and proof-of-workflow.

What it creates.

A JSON .vault file containing schema, KDF parameters, nonces, wrapped key material, and ciphertext.

Argon2id

Derives a wrap key from your phrase and a random salt.

XChaCha20-Poly1305

Authenticated encryption for key wrapping and content encryption.

AAD binding

Protected metadata is bound into the seal so meaningful tampering should fail.
