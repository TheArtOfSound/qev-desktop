QEV for Android

The same offline encryption vault that runs on Mac and Windows, now on your phone. Same vault format. Same crypto. Same phrase-based security. No accounts, no servers, no Play Store. Free and open.

Install

  1. Tap the download button above. Your browser will download QEV_0.28.1_android.apk.
  2. Open your phone's Downloads folder (or tap the notification when the download finishes).
  3. Tap the APK file. Android will ask you to allow installing from this source — tap Settings, enable "Install unknown apps" for your browser, then come back and tap Install.
  4. Open QEV from your app drawer. The first launch runs a crypto self-test (takes ~1 second). When the status reads Ready, the app is working.
  5. Encrypt a message, or open a .vault.json file someone sent you. The vault format is the same across Mac, Windows, Android, the web, and the CLI — a file encrypted on any platform decrypts on any other.
Sideload note: because the APK is not distributed through the Play Store, Android shows a sideload warning the first time. This is normal for any directly downloaded APK. The permission is per-browser — once granted, future updates install without the prompt.

What's included

  • XChaCha20-Poly1305 + Argon2id — same primitives as the desktop app
  • Crypto self-test on every launch (known-answer vector)
  • Mobile-optimised UI with 48px tap targets and safe-area insets
  • File-import intent filter — tap a .vault.json in your file manager or email and QEV offers to open it
  • Network security config blocking all cleartext traffic
  • Cloud backup and device-transfer explicitly disabled
  • QR pairing + direct P2P vault transfer (same LAN)
  • Relay-based offline delivery via secure.imagineqira.com
  • Same vault format (BRY-NFET-SX-VAULT-V2) as every other QEV platform

Other platforms

  • Mac + Windows desktop app — native installers
  • Web vault — browser-based, same crypto
  • Linux / CLI — npm i -g @imagineqira/qev-cli
