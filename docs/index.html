QEV

QEV — Qira Encryption Vault — is a local-first workflow for encrypted, tamper-evident vault artifacts. It is built for notes, AI output receipts, research artifacts, operational logs, and sensitive records that need a portable offline envelope.

Established primitivesXChaCha20-Poly1305, Argon2id, libsodium, and deterministic associated-data binding.
Local/offline firstNo account required for the CLI workflow. Lock and unlock files locally.
Honest scopeNot a new encryption algorithm, not a password manager, and not a substitute for security review.
